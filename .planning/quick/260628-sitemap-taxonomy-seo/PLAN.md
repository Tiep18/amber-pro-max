# Quick Task: Add taxonomy URLs to localized sitemap

## Goal

Ensure indexable category and collection landing pages are discoverable through localized sitemap files.

## Scope

- Add category URLs to `/sitemaps/[locale]`.
- Add collection URLs to `/sitemaps/[locale]`.
- Reuse cached catalog facets so sitemap generation does not add a new data path.
- Verify lint/build and sitemap output.

# Summary

Redesigned the admin product editor as a stable two-column scrollspy workspace.

## Changes

- Replaced collapsible sections with always-visible, priority-ordered form sections.
- Added synchronized section navigation: clicking the control rail scrolls to a section, while manual scrolling updates the active navigation item.
- Added Vietnamese and English tabs for content and SEO with locale-specific error counts.
- Reused the catalog Zod schema for client validation, inline field errors, automatic locale switching, scroll-to-error, and focus management.
- Replaced the duplicate sticky command bar with a viewport-height desktop control rail containing concise metadata, section navigation, workflow links, save, and publish actions.
- Added a mobile sticky section selector, shadcn Sheet outline, and safe-area-aware bottom action bar.
- Replaced native market checkboxes with the existing Toggle primitive and improved form control labels.
- Preserved catalog server actions, action payloads, cache invalidation, SEO, ISR, and storefront behavior.

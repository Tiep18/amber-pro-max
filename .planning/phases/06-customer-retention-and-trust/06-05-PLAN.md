## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser to review actions | Untrusted content, rating, and product ids cross into purchase eligibility. |
| Orders to review eligibility | Paid order-line evidence decides review permission. |
| Review identity to public UI | Customer identity must be transformed before public rendering. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-06-05-02 | Tampering | Customer review update | mitigate | Unique customer/product row and edit-to-pending state transition per D-09 and D-10. |
| T-06-05-03 | Information Disclosure | Public review projection/components | mitigate | Approved-only projection and masked public identity per D-11. |
| T-06-05-SC | Tampering | npm package surface | accept | No package install occurs. |
</threat_model>

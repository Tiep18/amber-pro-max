## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Email link to unsubscribe handler | Bearer token from email crosses into unauthenticated state mutation. |
| Outbox email rendering to subscriber state | Send-time unsubscribe URL must not persist raw token material. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-08-01 | Tampering | Unsubscribe token flow | mitigate | High-entropy raw token, SHA-256 hash-only storage, expiry/consumed metadata, and generic result states per D-14/D-16. |
| T-06-08-02 | Information Disclosure | Email/outbox/token rendering | mitigate | Raw token is used only in the outgoing URL, token hash never renders, and raw request metadata stays redacted. |
| T-06-08-SC | Tampering | npm package surface | accept | No package install occurs; reuse existing Resend/outbox stack. |
</threat_model>

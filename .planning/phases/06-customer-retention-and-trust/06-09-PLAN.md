## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Admin subscriber page to consent records | Privileged admin read crosses into subscriber status and consent history. |
| Consent metadata to admin UI | Sensitive metadata must be summarized without raw values. |
| Admin UI to consent state | Admin must not gain mutation paths for subscribe/unsubscribe. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-06-09-02 | Tampering | Admin consent override | mitigate | Do not implement override actions/RPCs/buttons; enforce with static security checks per D-15. |
| T-06-09-03 | Information Disclosure | Subscriber metadata | mitigate | Render full email only in protected admin list; never render raw IP, user-agent, raw token, or token hash per D-16. |

</threat_model>

## Trust Boundaries

| Boundary | Description |
|---|---|
| provider/admin/cron -> transition RPC | Commands affect money, inventory and future fulfillment eligibility. |
| transition RPC -> inventory/order/audit | All effects must commit or roll back together. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|---|---|---|---|---|
| T-04-02 | Tampering | inventory finalization | mitigate | Deterministic row locks, active-reservation predicate and conditional stock decrement. |
| T-04-03 | Tampering | replay/race | mitigate | Unique event/transition keys, monotonic matrix and typed duplicate/stale outcomes. |
| T-04-04 | Repudiation | money/stock changes | mitigate | Append-only audit with actor, source, facts, transition and before/after states. |
| T-04-SC | Tampering | package supply chain | accept | No package install; schema and existing pinned toolchain only. |

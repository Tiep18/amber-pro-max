## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser address form -> checkout action | Untrusted address and country selection cross into server quote/submit code. |
| locale/market/country selection | Customer-visible locale and market can diverge from shipping country. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-04 | Tampering | destination country | mitigate | Normalize ISO code server-side and make quote hash include destination-sensitive facts. |
| T-04-05 | Information Disclosure | digital-only checkout | mitigate | Do not ask for or persist address data when every payable line is digital. |
| T-04-06 | Denial of Service | country search | accept | Owned local list and client filtering are bounded, no external API call. |
| T-04-SC | Tampering | package installs | accept | No package install; country select uses owned country list and platform display names. |
</threat_model>

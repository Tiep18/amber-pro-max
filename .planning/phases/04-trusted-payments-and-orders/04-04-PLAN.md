<threat_model>
## Trust Boundaries

| Boundary | Description |
|---|---|
| browser -> PayPal routes | Order/provider IDs are untrusted and can target other orders. |
| server -> PayPal REST | Credentials, merchant identity and idempotent operations cross an external boundary. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|---|---|---|---|---|
| T-04-08 | Tampering | amount/currency/method | mitigate | Immutable local totals, intl+USD+PayPal invariant and exact provider reconciliation. |
| T-04-09 | Tampering | provider retries | mitigate | Persisted operation UUIDs, retrieve-after-uncertain and shared transition idempotency. |
| T-04-10 | Elevation of privilege | cross-order route access | mitigate | Signed-in owner or matching scoped guest cookie before provider I/O. |
| T-04-SC | Tampering | package supply chain | accept | Direct Web `fetch`; no PayPal npm dependency. |
</threat_model>

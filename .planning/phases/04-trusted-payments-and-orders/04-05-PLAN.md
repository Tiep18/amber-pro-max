## Trust Boundaries

| Boundary | Description |
|---|---|
| PayPal SDK callback -> customer UI | Client callback is not verified payment evidence. |
| customer route -> server projection | Route/order number can be used for IDOR/enumeration. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|---|---|---|---|---|
| T-04-11 | Spoofing | return/callback status | mitigate | Render only server projection; URL/query/callback cannot set paid. |
| T-04-12 | Tampering | client payment state | mitigate | Server create/capture routes and bounded recheck; no browser totals or direct transition. |
| T-04-13 | Information disclosure | guest order route | mitigate | Scoped HttpOnly cookie, generic denial and no token in URL/storage/props. |
| T-04-SC | Tampering | package supply chain | accept | Official hosted SDK; no npm install. |
</threat_model>

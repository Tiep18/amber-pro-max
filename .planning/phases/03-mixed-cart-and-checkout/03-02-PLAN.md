## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Admin form to shipping tables | Shipping rules affect payable totals and destination eligibility. |
| Destination input to quote | Shopper-controlled country can change market, currency, and availability. |
| Quote diff to accepted checkout state | Client must not apply material changes without explicit confirmation. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-03-02-01 | Tampering | shipping fees | mitigate | Integer money constraints, admin-only mutations, pgTAP negative cases. |
| T-03-02-02 | Tampering | destination quote | mitigate | Server recalculation and material diff before accepted-state change. |
| T-03-02-03 | Information Disclosure | unavailable lines | mitigate | Keep public line content visible but omit wrong-market price and payable action. |
| T-03-02-04 | Repudiation | admin shipping changes | mitigate | Store updated metadata; later audit event integration remains Phase 4/7 if global audit table is introduced. |

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser request -> `src/proxy.ts` | Untrusted path, headers, cookies, and query parameters influence locale redirects. |
| localized route map -> public URLs | Internal routes become externally indexable customer-facing paths. |
| test runner -> route contract | Automated tests become the guardrail for locked decisions D-01 through D-04. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-02-01 | Spoofing | `src/proxy.ts` locale negotiation | mitigate | Allow only `vi` and `en`; explicit URL locale wins; browser preference applies only to unprefixed first visits per D-01 and D-02. |
| T-01-02-02 | Tampering | language switch query preservation | mitigate | Preserve only allowlisted safe query keys and reject protocol-relative or external redirect targets. |
| T-01-02-03 | Information Disclosure | localized routing tests | accept | Route tests assert public URL behavior only and do not use secrets or personal data. |
</threat_model>

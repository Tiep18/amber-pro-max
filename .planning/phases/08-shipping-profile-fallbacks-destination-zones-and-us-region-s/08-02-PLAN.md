## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Quote intent -> public wrapper | Product, variant, quantity, destination, region, and currency are untrusted. |
| Public wrapper -> private resolver | SECURITY DEFINER ownership must not become a direct execution grant to browser roles. |
| Resolver -> calculator | Canonical fees/evidence must be same-currency, non-negative, and line-complete before aggregation. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-03 | Tampering / Denial of Service | Six-tier resolver | mitigate | Bound line intent, normalize destination, use authoritative joins, require one allocation per physical line, order deterministically, and fail closed. |
| T-08-04 | Tampering | Region/currency arithmetic | mitigate | Inherit selected-rule currency, constrain integer amounts/modes/codes, reject overflow/cross-currency data, and test SQL/TypeScript parity. |
| T-08-02-PRIV | Elevation of Privilege / Information Disclosure | Private resolver execution | mitigate | Revoke ALL from PUBLIC/anon/authenticated, grant only service_role direct execution, keep owner path implicit, expose a fixed-search_path constrained wrapper, and pgTAP both metadata and actual calls. |
| T-08-SC | Tampering | Dependency surface | accept | No package install or dependency is introduced. |

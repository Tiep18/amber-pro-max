## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser form to server actions | Product, slug, money, and taxonomy payloads are untrusted. |
| UI to publish RPC | UI presents state; database decides publishability. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-02-02-02 | Tampering | publish status | mitigate | Browser cannot set published; action uses atomic RPC. |
| T-02-02-03 | Information Disclosure | action errors | mitigate | Stable result codes; no raw database payloads. |

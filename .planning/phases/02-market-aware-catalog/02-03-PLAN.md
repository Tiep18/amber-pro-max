## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Uploaded bytes to Storage | MIME, size, path, and object ownership are untrusted. |
| Storage object to public UI | Images may be public; PDFs must never cross this boundary. |

| T-02-03-03 | Denial of Service | large uploads | mitigate | Bucket and server file-size limits. |
